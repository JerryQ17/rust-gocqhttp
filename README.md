## 简介

`rust-gocqhttp`是一个基于Rust语言，为[`go-cqhttp`](https://github.com/Mrs4s/go-cqhttp)开发的SDK。

所有的`CQCode`，`API`的命名均与`go-cqhttp`原生实现保持一致。

## 支持

### 通信协议
